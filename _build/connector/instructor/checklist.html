Pre-Semester Checklist

Administrative

  • Schedule a class time
  • Request rooms for class, office hours
  • Request student help: student developers, Connector Assistants (CAs), graders
  • Develop your syllabus
  • Get access to bCourses if you would like to use it

Technical

  • Read through this guide
  • Create/update public and private GitHub repos for your course in the data-8 organization
  • Create/update your course website
  • Sign up for connector instructor Piazza
  • Post on Piazza if you need any Python packages installed on DataHub
